# Base SQL

Les fichiers de ce dossier décrivent la base de données relationnelle du projet.

- `schema.sql` contient les tables, les clés étrangères et les index.
- `data.sql` contient un jeu de données initial pour lancer l'application rapidement.

Le modèle couvre les formations, les étudiants, les contenus pédagogiques, le syllabus, les inscriptions, les paiements et les notifications.
